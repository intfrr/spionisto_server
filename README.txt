Spionisto server software
===========================

--------

Please, do read the LICENSE.txt file to find out about restricted use of this

DESCRIPTION
------------

Spionisto server provides a web interface and a backend server for the
Spionisto surveillance software.

The web interface is written in the Pyramid framework and stores its settings
using the ZODB database.

The backend server is written in plain Python and requires and
gstreamer-python bindings.

INSTALLATION
-------------

easy_install spionisto_server
